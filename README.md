# Calculadora de Potências em C

Projeto desenvolvido durante meus estudos iniciais de lógica de programação utilizando a linguagem C.

## Objetivo

O objetivo deste projeto foi praticar conceitos fundamentais da programação em C, como:

* Entrada e saída de dados (`scanf` e `printf`)
* Uso de variáveis inteiras
* Operações matemáticas básicas
* Lógica de programação
* Repetição manual de cálculos
* Estrutura sequencial de execução

## Sobre o projeto

Este programa funciona como uma calculadora simples de potências.

O usuário informa um número inteiro e o programa calcula manualmente suas potências de 2 até 10.

Exemplo:

Se o usuário digitar `2`, o programa irá calcular:

* 2²
* 2³
* 2⁴
* ...
* 2¹⁰

E exibir todos os resultados no terminal.

## Como funciona

O programa:

1. Solicita que o usuário digite um número
2. Armazena esse valor em uma variável
3. Calcula manualmente cada potência (de 2 até 10)
4. Exibe os resultados na tela

Cada potência é calculada através de multiplicações sucessivas, sem uso de funções matemáticas externas.

Este projeto foi feito em fase inicial de aprendizado, por isso os cálculos foram feitos de forma manual e repetitiva, sem uso de loops ou funções matemáticas.

Isso foi proposital para reforçar a compreensão da lógica por trás das operações.
